# API Keys Configuration

## Riot API Setup

### Получение API ключа:
1. Перейди на https://developer.riotgames.com/
2. Авторизуйся через аккаунт Riot
3. Создай новый API ключ в разделе "My Applications"
4. Для локальной разработки добавь ключ в `.env`

### Правила использования Riot API:
- **Rate Limits**: 1 запрос в секунду для разработческих ключей
- **Цели использования**: Только для личного использования и тестирования
- **Запрещено**: Коммерческое использование без разрешения Riot Games
- **Требования**: Добавление credit к Riot Games в интерфейсе

### Переменные окружения:
```env
VITE_RIOT_API_KEY=RGAPI-your-key-here
```

⚠️ **ВАЖНО**: Никогда не коммить реальные API ключи в Git и не давай доступ к ним через клиентский интерфейс.

## Безопасность

### Для production:
1. Используй серверный прокси для API запросов
2. Не храни API ключи на клиенте
3. Добавь аутентификацию пользователей
4. Реализуй кэширование для снижения нагрузки на API

### Локальная разработка:
- `.env` файл игнорируется в Git
- Каждый разработчик создаёт свой `.env` из `.env.example`
- Пользовательский ввод Riot API ключа в приложении отключен
