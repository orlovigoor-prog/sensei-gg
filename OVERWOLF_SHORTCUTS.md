# Настройка горячих клавиш в Overwolf

## 🎯 Глобальные горячие клавиши

Overwolf позволяет настроить глобальные горячие клавиши для управления приложением даже когда League of Legends активна.

### Способ 1: Через настройки Overwolf (рекомендуется)

1. **Откройте настройки Overwolf:**
   - Нажмите `Ctrl + Shift + O` в любом месте
   - Или кликните на иконку Overwolf в трее → ⚙️ Настройки

2. **Перейдите в раздел "Приложения":**
   - Найдите **Sensei GG** в списке приложений
   - Нажмите на значок ⚙️ рядом с названием

3. **Настройте горячие клавиши:**
   - В разделе **"Горячие клавиши"** найдите опцию **"Открыть/закрыть приложение"**
   - Нажмите на поле и введите желаемую комбинацию (например, `Ctrl + X`)
   - Нажмите `Enter` для сохранения

4. **Проверьте настройки:**
   - Запустите League of Legends
   - Нажмите настроенную комбинацию — окно Sensei GG должно открыться/закрыться

### Способ 2: Через manifest.json (для разработчиков)

В `manifest.json` можно задать стандартные горячие клавиши:

```json
{
  "overwolf_apis": [
    "Windows",
    "KeyboardShortcuts"
  ],
  "keyboard_shortcuts": {
    "toggle_window": {
      "default_combination": {
        "key": "X",
        "ctrl": true,
        "alt": false,
        "shift": false
      },
      "description": "Открыть/закрыть окно приложения"
    }
  }
}
```

## ⌨️ Поддерживаемые клавиши

### Модификаторы:
- `Ctrl` (Control)
- `Alt`
- `Shift`
- `Win` (Meta)

### Основные клавиши:
- `A-Z` (латинские буквы)
- `0-9` (цифры)
- `F1-F12` (функциональные)
- `Space` (Пробел)
- `Escape` (Esc)
- `Enter`
- `Arrow*` (Стрелки)

### Примеры комбинаций:
- `Ctrl + X` — Открыть/закрыть (по умолчанию)
- `Ctrl + Shift + S` — Быстрая статистика
- `Alt + S` — Открыть Sensei
- `Ctrl + F12` — Полноэкранный режим

## 🔧 Решение проблем

### Проблема: Горячие клавиши не работают в игре

**Причина:** League of Legends перехватывает нажатия клавиш

**Решение:**
1. Используйте комбинации с `Ctrl + Shift` или `Alt`
2. Избегайте клавиш, используемых в игре (Q, W, E, R, D, F)
3. В настройках Overwolf включите **"Always capture keyboard"**

### Проблема: Окно не открывается

**Решение:**
1. Проверьте, что приложение запущено (иконка в трее)
2. Нажмите `Ctrl + Shift + D` для открытия dev-панели
3. Проверьте логи в консоли (F12)

### Проблема: Конфликт с другими приложениями

**Решение:**
1. Измените комбинацию на уникальную
2. Отключите горячие клавиши в конфликтующих приложениях
3. Приоритет у Overwolf: Settings → Apps → Sensei GG → Priority

## 📱 Горячие клавиши внутри приложения

### Встроенные комбинации:
- `Ctrl + X` — Свернуть/развернуть оверлей
- `Ctrl + Shift + D` — Открыть dev-панель (только в dev режиме)

### Настройка через интерфейс:

1. Откройте вкладку **⚙️ Настройки**
2. Найдите раздел **"Горячие клавиши"**
3. Нажмите на кнопку с текущей комбинацией
4. Введите новую комбинацию
5. Нажмите **"Сохранить"**

## 🎮 Рекомендации для League of Legends

### Рекомендуемые комбинации:
```
✅ Ctrl + X         — Открыть/закрыть оверлей
✅ Ctrl + Shift + D — Dev-панель в локальной разработке
```

### Избегайте:
```
❌ Q, W, E, R    — Способности чемпиона
❌ D, F          — Зелья/исчезновение
❌ Space         — Приостановка камеры
❌ B              — Отход в базу
❌ Ctrl + Click   — Автоатака
```

## 🔄 Обновление настроек

Если настройки сбрасываются:

1. Проверьте файл `localStorage` в браузере
2. Убедитесь что нет ошибок в консоли
3. Попробуйте сбросить и настроить заново
4. В крайнем случае — переустановите приложение
