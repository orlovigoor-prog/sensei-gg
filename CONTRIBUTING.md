# Contributing to Sensei GG

Thank you for your interest in contributing! This project is primarily for personal and educational purposes, but contributions are welcome.

## 🤝 How to Contribute

### Reporting Bugs
- Check existing issues first
- Provide detailed description
- Include steps to reproduce
- Add screenshots if applicable

### Suggesting Features
- Open an issue with feature request
- Describe the use case
- Explain benefits for users

### Code Contributions
1. Fork the repository
2. Create feature branch (`git checkout -b feature/amazing-feature`)
3. Commit changes (`git commit -m 'Add amazing feature'`)
4. Push to branch (`git push origin feature/amazing-feature`)
5. Open Pull Request

## 📝 Code Style

- Use TypeScript for all new code
- Follow existing patterns
- Add comments for complex logic
- Keep components focused

## ⚠️ Important Notes

- This is a personal project for learning
- Don't commit API keys
- Respect Riot Games API policies
- Test changes before submitting

## 🚫 What Not to Contribute

- Commercial features
- Data storage solutions
- Anything violating Riot Games ToU

Thank you for helping improve Sensei GG! 🎮
